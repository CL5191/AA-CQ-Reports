# SLICE-0011 CI Report Artifacts

## Goal

Publish generated AA/CQ sample reports as build artifacts in CI so stakeholders can download outputs from each run.

## Acceptance Criteria

- CI continues to run lint, test, and build checks.
- CI generates report files from sample CQ and AA datasets.
- CI uploads generated files as an artifact for each run.
- README documents artifact naming and included files.
